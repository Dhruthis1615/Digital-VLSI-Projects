# Digital VLSI Projects

This repository contains a collection of **Digital VLSI design projects** implemented using **Verilog HDL** and simulated using **Cadence Virtuoso**. These projects were developed to strengthen understanding of digital hardware design and system architecture.

The projects demonstrate the design and implementation of fundamental **digital hardware components**, including arithmetic units, memory modules, processor pipelines, and digital signal processing blocks.

---

## Repository Structure

```
Digital-VLSI-Projects
│
├── Project 1 - ALU
├── Project 2 - RAM design
├── Project 3 - Pipeline Processor Design
└── Project 4 - Digital filter Design
```

Each project folder contains the necessary design files and implementation details related to that module.

---

## Projects Included

### 1. Arithmetic Logic Unit (ALU)
Design and implementation of an **Arithmetic Logic Unit** capable of performing arithmetic and logical operations. The ALU forms the core computational unit of most processors.

### 2. RAM Design
Implementation of a **Random Access Memory (RAM)** module using Verilog HDL with controlled read and write operations.

### 3. Pipeline Processor Design
Design of a **pipelined processor architecture** demonstrating instruction execution across multiple pipeline stages such as instruction fetch, decode, execute, memory access, and write-back.

### 4. Digital Filter Design
Implementation of a **digital filter** architecture used in digital signal processing applications.
